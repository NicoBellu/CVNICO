# Portfolio Junior Developer

Questo repository contiene tre progetti realizzati come portfolio per candidarmi come Junior Developer. Tutti i progetti sono sviluppati utilizzando HTML, CSS e JavaScript con un design semplice e minimale.

## Progetti

### 1. Animazione Testo (ANIMAZIONE)

* **Descrizione:** Una pagina che mostra una parola con animazioni sequenziali sulle singole lettere.
* **Funzionalità:**

  * Ogni lettera ha un colore unico.
  * Animazioni: rimbalzo, rotazione, caduta.
  * Le animazioni partono in sequenza e il ciclo si ripete all'infinito.
* **Tecnologie:** HTML, CSS, JavaScript.
* **Come usarlo:** Aprire `ANIMAZIONE.html` nel browser.

### 2. Gioco Interattivo (Click-game)

* **Descrizione:** Un gioco dove l'utente deve cliccare un quadrato che cambia posizione ogni tot secondi.
* **Funzionalità:**

  * Il quadrato cambia posizione automaticamente ogni 1,5 secondi.
  * Il punteggio viene incrementato solo quando il quadrato viene cliccato.
  * Pulsanti per avviare/pausare e resettare il gioco.
* **Tecnologie:** HTML, CSS, JavaScript.
* **Come usarlo:** Aprire `Click-game.html` nel browser e cliccare su "Avvia".

### 3. Simulazione Login Sicurezza (Login-simulation)

* **Descrizione:** Una demo di login che simula controlli di sicurezza sulla password.
* **Funzionalità:**

  * Controlla che la password abbia almeno 8 caratteri, includa lettere maiuscole, numeri e simboli.
  * Fornisce feedback immediato all'utente.
  * Dopo 3 tentativi falliti, il login viene bloccato per 30 secondi con countdown.
  * Pulsante per tornare alla Home.
* **Tecnologie:** HTML, CSS, JavaScript.
* **Come usarlo:** Aprire `Login-simulation.html` nel browser.
